Introduction

The gameboy is a handheld gaming console released in the late 80's. Emulators now exist which can run games including new games developed entirely for that virtual system (or for die-hard fans with money to burn, small-batch manufactured games). This means that the gameboy is an excellent vehicle for learning about low level, resource constrained, and single-purpose systems. I use and recommend the RGBDS toolchain for assembling code, Tilemap studio and Pixelorama for generating pixel art, and finally VIM and geany for programming.

The GB Dev community (especially the discord chat) is incredibly valuable in working through this process. I strongly recommend anyone interested in the topic to start there.
